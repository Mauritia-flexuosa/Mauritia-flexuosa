### Olá! Eu sou o Marcio Cure.  /  Hi! My name is Marcio Cure. 👋

- 🔭 I hold a Bachelor's degree in Environmental Management, along with a Master's and a PhD in <a href="https://www.poseco.ufsc.br">Ecology</a>.
